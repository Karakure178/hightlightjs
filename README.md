# これは何？
hightlight.jsをvueで使うためのメモ

実際に使ってみて、タブレイアウトと切り替えて遊べるようにするところまで。

## 実行コマンド
| Command                   | Action                                           |
| :------------------------ | :----------------------------------------------- |
| `npm install`             | 環境のインストール                            |
| `npm ci`             | バージョンを固定してインストール                            |
| `npm start`             | サーバーの立ち上げ：`localhost:3001`      |
